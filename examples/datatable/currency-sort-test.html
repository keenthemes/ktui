DataTable – Currency sort

Price column uses sortType: 'numeric'. Click the Price (£) header and confirm order is by value: £5, £20, £123 (not string order like £123, £20, £5).

Without numeric sort, "£123" would appear before "£20" because "1" < "2" lexicographically.
